# 🧪 Java Lab / Learning Path

Este repositório foi criado para armazenar meus códigos de estudos, exercícios e desafios práticos em **Java**. 
O objetivo principal é documentar a minha evolução e a aplicação de conceitos de Orientação a Objetos (POO).

## 🚀 O que eu já aprendi e apliquei por aqui:

* **Interfaces e Polimorfismo:** Criação de contratos de código e redefinição de comportamentos com `@Override` (visto no desafio dos Ninjas).
* **Classes Abstratas:** Uso de classes que servem como moldes genéricos (como a classe `Hokage`) e herança com a palavra-chave `extends`.
* **Construtores e Reaproveitamento:** Inicialização de atributos direto na criação do objeto e uso do `super()` para invocar construtores da classe pai.
* **Enums:** Uso de estruturas para limitar e organizar opções fixas (como os tipos de jutsus/habilidades).
